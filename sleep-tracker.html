Visualizing Sleep Data

Altering information visualization influence through sleep feedback.

Timeline:

Dec 2022

Role:

UI/UX Designer

Project Type:

Course Project

Course Introduction + Prefaces

This project was the final undertaking of the course CLPS 1580c: Visualizing Information, taught by Professor Leslie Welch at Brown University. The course explores how to effectively present information in a visual way through examining the current state of the field of information visualization and infographics and drawing on principles ranging from graphic design to the neurological visual system.

Prompt

There were three options for this project, and I chose the one that would show three different aspects of the same information, but was intended for the same audience and context/format.

Design

Inspiration

I found that a daily place where information visualization was extremely prevalent was health-tracking apps. The recent increased emphasis on health made me wonder how our trust and habits can be dictated by the way that these apps visualize that information, and I was inspired by Fitbitâ€™s app, which I used to use regularly.

Their sleep page is as follows, telling you all about how long you slept, how long you spent in each sleep stage, what each sleep stage is, and eventually scoring it:

Ideation

I wanted to convey sleep data, specifically duration, to the public user in 1) a neutral manner that simply relayed the numerical information to the user, 2) a scolding manner that reprimanded the user to get more sleep, and 3) a rewarding manner that justified the user regardless of the hours.

Results

Neutral

This prototype simply conveyed the number of hours slept.

Scold

This prototype juxtaposed hours of sleep to executive function to indicate lower function with less sleep, as drawn from this study.

Reward

This prototype justified the low hours of sleep by instead prioritizing the quality of the sleep.
